Favicon:
-> A favicaon that stands for favorite icon is a tiny include along a website which is displays in a places like a browser's address bar, page tabs, and book mark menu. It is usually 16*16 pixel in size and stored in GIF, PNG, or ICO File format. Favicon provide a visual representation of a website, helping users quickly identify and locate it among multiple open tabs or bookmarks.They are used to improve user experience and enforce brand consistencty. When a familiar icon is seen in the browser address bar.
